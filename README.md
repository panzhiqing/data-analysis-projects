# Data Analysis Projects

This repository contains selected data analysis projects implemented using SQL, Python and machine learning techniques.

## Projects

### 1. Predicting Next-Day Total Sales
- Objective: Predict next-day total sales using historical transaction data.
- Techniques: Exploratory Data Analysis, baseline modeling, MAE evaluation.
- Tools: Python, Pandas, Seaborn, Matplotlib.

### 2. Telco Customer Churn Analysis
- Objective: Identify key drivers of customer churn.
- Techniques: Logistic Regression, Random Forest, AUC evaluation.
- Tools: Python, Pandas, Scikit-learn, SQL-style aggregations.

These projects are designed as portfolio examples for entry-level data analyst roles.
