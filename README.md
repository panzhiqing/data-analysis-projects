# Data Analysis Projects

This repository contains selected data analysis projects implemented using Python, SQL, and machine learning techniques.

## Projects

### 1. Predicting Next-Day Total Sales
- Objective: Predict next-day total sales using historical transaction data.
- Techniques: Exploratory Data Analysis, feature engineering, baseline modeling.
- Tools: Python, Pandas, Matplotlib.

### 2. Telco Customer Churn Analysis
- Objective: Identify key drivers of customer churn and build predictive models.
- Techniques: EDA, Logistic Regression, Random Forest, AUC evaluation.
- Tools: Python, Pandas, Scikit-learn, SQL-style aggregations.

These projects are designed as portfolio examples for entry-level data analyst roles.
